## 1. Introduction

### 1.1 Project Overview

Air Quality Index (AQI) is a critical environmental indicator that measures pollution levels and their impact on public health. This project focuses on analyzing AQI data across Indian states and cities to monitor pollution trends, identify hazardous zones, and understand pollutant contribution using **Power BI**.

The objective is to convert raw environmental data into an interactive and insight-driven dashboard that helps policymakers, researchers, and citizens make informed decisions regarding air quality management.

---

## 1.2 Objectives

The primary goals of this project are:

* Analyze overall AQI performance by state and city
* Identify primary pollutants contributing to poor air quality
* Classify air quality into standard AQI categories
* Track AQI trends over time (Year & Month analysis)
* Compare pollution levels across different regions
* Enable dynamic filtering for interactive insights
